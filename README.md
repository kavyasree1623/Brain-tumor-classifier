# Brain Tumor Classifier 🧠

This project classifies brain tumor MRI images using deep learning.

## 📂 Project Structure
brain/
│── app.py
│── pos.py
│── checkpoint.weights.h5
data/ (not included)

## 🚀 How to Run
1. Clone the repository

git clone https://github.com/YOUR_USERNAME/brain-tumor-classifier.git
Install dependencies

pip install -r requirements.txt
Run the app

python brain/app.py

📊 Dataset
The dataset is not included due to size limitations.
Use any brain tumor MRI dataset (e.g. Kaggle).

🧠 Model
CNN-based deep learning model trained on MRI images.
